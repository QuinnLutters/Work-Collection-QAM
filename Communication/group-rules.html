Group rules

Groups need rules, we found out the hard way. When you have a group project coming up, making a group rules document should be one of the first things you do, since it makes everyone keep in line. The most important thing I that group rules document is that there should be some kind of punishment if you don’t listen to the rules, for example what we had was if you have three strikes you’ll have to buy everyone a drink the next day. This allows people to actually listen, since there is something on the line. Due note if you have a system like that, use it. We didn’t use it as we should, which might be one of the factors, which hindered our progress.
