Color selector

To start of JavaScript we had to make a color selector which allowed a user to change the color of a website on the fly. This actually was relatively easy using query selector and making a small calculation to change a variable, which changed the variable of a root: selector in css. I also made this but by changing the font size of an element and the height of an element by changing their variables. This can be really useful when your working with a client, since this allows you to show them your website and allow them to change little change live on the website.
